# To Do List (Hooks, Firebase Cloud Database)

### [Live Version Of Website](https://todolist-78f6b.web.app/)

##### This is a Todo List Application developed using React JS and Firebase Firestore. The Project has 4 Pages: A root page, a Todo Page a About page and a 404 error Page. The App was made by me (William Simms).
